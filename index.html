Projects

Notable work

Prototyping a Functional Hearing Aid

Final year research project. Used MATLAB to evaluate the algorithms and C to implement the evaluated algorithms on the DSP TMS320C5535 eZDSP development board using Code Composer Studio.

Twitter Sentiment Analysis

This project focuses on Sentiment Analysis on a twitter dataset provided by Kaggle. This project was done as a partial fulfillment of the course Twitter Sentiment Analysis provided by Coursera. The project uses Pandas, Numpy, Matplotlib and Seaborn libraries to analyze and visualize data taken from a .csv file.

Twitter K-Means Clustering

This project focuses on analysing tweets and clustering them using K-Means clustering to get ideas on tweets. Here, the main metrics used for clustering are Elbow Method and Silhouette Method. In addition, MySQL is used to get the .csv file from an .sql dump file. Here, Numpy, Pandas, Sklearn, Nltk, Matplotlib, Seaborn and several other libraries have been used to clean and lemmatize the data. Finally, the clusters are shown using Wordnet implementation.

Language Classification with Naive Bayes

This project was done as a partial fulfillment of the course provided by Coursera Here, we are trying to understand the words provided are whether belong to English, Czech or Slovak. Here, Zipf's model, Naive Bayes algorithm and the application of confusion matrices are used to complete the given task. Numpy, Sklearn and Pickle libraries are used in here with Python.

Box Office Data Analyzer

This project was done as a partial fulfillment of the course provided by Coursera. Here, we are trying to analyze and visualize box office data with Python. Here, Numpy and Pandas Librareis are used to analyze the Data Frames.

Ristorante Con Fusion

This project was done in Angular as well as React using Redux for the partial fulfillment of the two Coursera courses I've followed duruing my leisure time. The projects were done using Typescript and Javascript respectively. A web app for restaurant is made in both of the projects.

AGV Management System

This project was done during my internship at Sanota.Pvt.Ltd. This is a Java based application made with Gradle integration for factory automation. Here this application works as a UI for Automatic Guided Vehicle control which is equippd with an Arduino Mega microcontroller. The AGV application transmits the signals to an USB ESP8266/Node MCU chip and the AGV captures it via WiFi transmission.

Bleach - Payroll System

This project was done for partial fulfillment of an external course (IJSE-CMJD) I followed after A/Ls. The project was made using Java and MySQL while using Jasper for report generation. The MVC architecture was used here coupled with Singleton design pattern for database connectivity.

Cifer - Encryption Tool

Cifer is a simple file encryption tool I made using Java.

FIR Filter Implementation with Kaiser Windowing

This is a project we did for the partial fulfillment of the module Digital Signal Processing, which was offered during our fifth semester. The goal here was to determine the FIR filter implementaion using Kaiser windowing.

Image Downsampling Processor Design

This is a group project we did for the Circuits and System Design module which was offered in fifth semester. Here, we designed a processor using a Cyclone IV FPGA and Quartus Prime Software while simulating with Modelsim Altera. We used MATLAB for the image processing part.
